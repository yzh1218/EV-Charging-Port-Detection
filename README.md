# EV Charging Port Detection Based on YOLOv8

## Project Overview

This project is my undergraduate graduation project.

The goal is to detect charging ports of new energy vehicles using the YOLOv8 object detection framework.

## Main Work

- Built and organized the charging port image dataset
- Learned the YOLOv8 training workflow
- Performed image annotation and data augmentation
- Learned model training with PyTorch
- Learned Raspberry Pi deployment based on ONNX Runtime

## Tech Stack

- Python
- YOLOv8
- PyTorch
- OpenCV
- ONNX Runtime
- Raspberry Pi

## Application

New Energy Vehicle

Computer Vision

Object Detection
